# Task 2 - Image Encryption Tool

## Objective
Develop a simple image encryption tool using pixel manipulation techniques. The program encrypts and decrypts images using a user-defined key.

## Features
- Encrypt image pixels using a mathematical operation.
- Decrypt encrypted images using the same key.
- Supports PNG, JPG, and other common image formats.
- Simple command-line interface.

## Technologies Used
- Python 3
- Pillow (PIL)

## Installation

Install the required library:

```bash
pip install pillow
```

## Usage

Run the program:

```bash
python image_encryption.py
```

Choose:
- E for Encryption
- D for Decryption

Enter:
- Image file path
- Encryption/Decryption key
